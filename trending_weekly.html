🏆 项目一：mattpocock/skills — AI 编程技能集

⭐ 57.2k Stars | 📈 周增 34,848

核心亮点

这是由资深工程师 Matt Pocock 打造的 Claude/Codex AI 编程技能集，旨在解决 AI 编码代理的三大常见失败模式：

  1. 代理没按预期执行 → 提供 /grill-me 命令进行需求对齐
  2. 代理过于冗长 → 通过共享语言文档精简沟通
  3. 代码无法运行 → 小步迭代 + 快速反馈机制

技术架构

# 一键安装
npx skills@latest add mattpocock/skills

# 设置技能
/setup-matt-pocock-skills

实战价值

  • ✅ 与任何 LLM 模型兼容
  • ✅ 基于数十年工程经验
  • ✅ 模块化设计，易于定制
  • ✅ 支持 GitHub/Linear/本地文件 issue 追踪

适用场景

场景 技能命令 效果
需求对齐 /grill-me 代理主动提问，确保理解一致
代码审查 /triage 自动分类问题优先级
文档生成 /grill-with-docs 生成 ADR 和技术文档

💰 项目二：TradingAgents — 多智能体金融交易框架

⭐ 65.5k Stars | 📈 周增 11,252

核心亮点

这是一个 多智能体 LLM 金融交易框架，通过多个 AI 代理协作完成：

  • 市场分析
  • 策略制定
  • 风险管控
  • 自动执行

技术架构

# 示例：启动交易代理
from trading_agents import TradingSystem

system = TradingSystem(
    models=["gpt-4", "claude-3"],
    strategies=["momentum", "mean_reversion"],
    risk_management="dynamic"
)

system.run()

市场价值

  • 📊 支持多市场数据接入
  • 🤖 多模型协作决策
  • 📈 实时风险管控
  • 💡 可解释 AI 交易逻辑

🎁 项目三：free-claude-code — 免费 Claude Code 方案

⭐ 20.8k Stars | 📈 周增 8,276

核心亮点

让开发者在 终端/VSCode/Discord 中免费使用 Claude Code，支持语音交互。

安装方式

# 终端安装
npm install -g free-claude-code

# VSCode 扩展
ext install alishahryar.free-claude-code

功能特性

  • ✅ 终端模式：命令行交互
  • ✅ VSCode 集成：编辑器内使用
  • ✅ Discord 机器人：团队协作
  • ✅ 语音支持：语音编码
